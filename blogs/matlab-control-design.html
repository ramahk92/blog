MATLAB Control System Design

Simulation • Analysis • Controller Design

Author: Ram Kumar
Assistant Professor, Department of Electrical Engineering

Introduction

MATLAB is one of the most widely used software packages for analyzing and designing control systems. Engineers use MATLAB to model dynamic systems, analyze system stability, tune controllers, and simulate system responses before implementing them in real hardware.

Why MATLAB? MATLAB provides powerful built-in functions for transfer functions, state-space models, PID tuning, root locus, Bode plots, Nyquist plots, and Simulink-based simulations.

Features of MATLAB for Control Systems

  • Transfer Function Modeling
  • State Space Representation
  • Step Response Analysis
  • Impulse Response
  • Root Locus Design
  • Bode Plot
  • Nyquist Plot
  • Nichols Chart
  • PID Controller Design
  • Simulink Simulation

Example 1: Creating a Transfer Function


clc
clear
close all

num=[1];

den=[1 3 2];

G=tf(num,den)

step(G)

grid on

The tf() command creates the transfer function model, while step() displays the unit step response.

Example 2: Root Locus


clc
clear

G=tf([1],[1 4 5 2]);

rlocus(G)

grid on

title('Root Locus')

Root locus is used to determine how the closed-loop poles move as the proportional gain changes.

Example 3: Bode Plot


clc
clear

G=tf([5],[1 6 11 6]);

bode(G)

grid on

margin(G)

The Bode plot helps determine gain margin, phase margin, and system stability.

Example 4: PID Controller Design


clc
clear

s=tf('s');

G=1/(s*(s+2));

C=pid(5,2,1);

T=feedback(C*G,1);

step(T)

grid on

The pid() function creates a PID controller, while feedback() forms the closed-loop system.

MATLAB Functions Used in Control Engineering

Function Purpose
tf() Create Transfer Function
ss() State Space Model
step() Step Response
impulse() Impulse Response
rlocus() Root Locus
bode() Bode Plot
nyquist() Nyquist Plot
nichols() Nichols Chart
pid() PID Controller
feedback() Closed Loop System

Conclusion

MATLAB has become the industry-standard software for control system analysis and design. Its powerful toolboxes and graphical capabilities allow engineers and students to model, analyze, simulate, and optimize dynamic systems efficiently before real-world implementation.
